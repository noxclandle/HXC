# Hexa Card: NTAG215 発行・配布手順書

運営スタッフが物理カードを新規発行し、ユーザーに届けるまでの標準フローです。

## 1. 物理カードの準備
- 使用チップ: **NTAG215** (必須)
- NFC書き込みアプリ（NFC Tools等）を起動。

## 2. システム台帳への登録
1. 管理者画面の **[Ledger (台帳)](http://localhost:3001/admin/ledger)** を開く。
2. チップ固有の **UID** (例: `04:A2:3F:...`) と、管理用の **Serial Number** を入力し、[Create Slot] を実行。
3. これにより、システム側でそのカードの「受け入れ準備」が完了します。

## 3. カードへのデータ書き込み
- 以下のURLを「URIレコード」としてNTAGに書き込みます。
- **書き込みURL**: `https://hexa-card.com/api/card/[そのチップのUID]`
- ※ これにより、ユーザーがタップした際に自動的に認証・登録フローが開始されます。

## 4. ユーザーへの配布
- 書き込みが完了したカードをユーザーに手渡します。

## 5. ユーザーによる初期設定
1. ユーザーがカードをスマホでタップ。
2. 自動的にブラウザが開き、Identity Setup画面が表示される。
3. ユーザーが自身の情報を入力し、[Finish Setup] を押して完了。

---
## トラブル対応
- **カード紛失時**: 台帳画面から該当UIDを [Suspend (一時停止)] してください。ログインと名刺表示が即座に遮断されます。
- **発見時**: [Resume (再開)] ボタンで即座に復旧可能です。
